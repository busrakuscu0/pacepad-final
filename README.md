# PacePad

A mobile-responsive and modern task/focus management interface designed to simplify time tracking and boost productivity.

## 🚀 Features

- **Mobile Responsiveness:** Flawless display across both desktop and mobile devices using modern CSS architecture.
- **Task Tracking:** Dynamic mission cards displaying remaining time, custom tags, and status (e.g., Deep-Work, Urgent).
- **Clean UI:** A distraction-free, modern user interface built with customized checkboxes and focus badges.
- **Interactive Elements:** Smooth user experience and dynamic interactions powered by JavaScript.

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox, CSS Grid)
- JavaScript (Vanilla)

## 📦 Installation & Setup

To get a local copy up and running, follow these simple steps:

1. Clone the repository:
   ```bash
   git clone [https://github.com/busrakuscu0/pacepad-final.git](https://github.com/busrakuscu0/pacepad-final.git)
   ```
2. Navigate to the project directory:

   ```bash
   cd pacepad-final
   ```

3. Open the project:
   Simply locate the `index.html` file in the folder and open it with your favorite web browser. No extra installation or build process is required!
